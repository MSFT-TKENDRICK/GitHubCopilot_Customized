# Delayed shipment response guide

Use this guide when a customer ask why an order has not moved for more than 24 hours.
It is meant for fast triage, so it intentionally leave out the full carrier playbook.

## Before you reply

Confirm the tracking page was updated in the last 24 hours and that the newest carrier scan is visible.
If the order ship in multiple boxes, do not promise that every package will arrive together unless the carrier note says that clearly.
Use the saved reply below only when the delivery note does not mention a weather delay.

```powershell
npm run build
npm run test
```

## When to escalate

Escalate the case when the customer has already contacted support twice, the carrier has not scanned the parcel for more than 48 hours, or the warehouse note are missing.
After the call, add a short summary that explain what was checked and what the next agent still need to do.
